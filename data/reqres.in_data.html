Flows

Switch between authentication, CRUD, and pagination.

Scope data with Authorization: Bearer <session_token>.

Step 1Send magic link 
const login = await fetch('https://reqres.in/api/app-users/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': '<project_api_key>',
    'x-reqres-env': 'prod'
  },
  body: JSON.stringify({
    email: 'user@example.com',
    project_id: '<project_id>'
  })
}).then(r => r.json())
Step 2Verify magic token 
const verify = await fetch('https://reqres.in/api/app-users/verify', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ token: login.data.token })
}).then(r => r.json())

const sessionToken = verify.data.session_token
Step 3Pass the session token 
const authHeaders = {
  Authorization: 'Bearer ' + sessionToken
}

// send with any fetch()/client
await fetch('https://reqres.in/app/collections/profile/records', {
  headers: authHeaders
})
